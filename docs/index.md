## About Read logger tools

Read logger tools は動物搭載型の行動記録計（データロガー）で取得したデータをIgor proで読み取るためのパッケージです。
行動記録計では複数のデータが、バラバラのサンプリング間隔で記録されるので、手動でいちいち読み込むのは大変です。
このパッケージは、各データロガーで得られた生データの読み取りからデータの時刻設定までを半自動的に行います。
